# Data Lovers de Rick & Morty en React

## [Demo](http://merunga.github.io/data-lovers-rick-and-morty)

## Temas

* `create-react-app`
* Componentes funcionales (y mensión a componentes clase)
* `children` y `keys`
* `props` y `prop-types`
* `useState`
* `useEffect`
* CSS modules
* Uso de librerias de componentes externas (`react-bootstrap`)
* Manejo de errores
* `eslint`
* deploy a `gh-pages`
